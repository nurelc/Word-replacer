# Replace Words Project 📝

Этот проект показывает, как можно автоматически заменять слова в тексте с помощью Python.  

Программа работает очень просто:  

1. Пользователь вводит строку текста, например:  
   `"Я люблю Python"`  
2. Затем он указывает слово, которое хочет заменить, например `"Python"`.  
3. Далее вводится слово, на которое нужно заменить, например `"Java"`.  
4. Скрипт ищет все вхождения указанного слова в тексте и заменяет их на новое.  
5. В итоге пользователь получает готовую строку с заменой:  
   `"Я люблю Java"`  

Таким образом программа позволяет быстро и легко менять слова в любом тексте, будь то предложение, стих или даже несколько строк.  

Принцип работы основан на **проходе по тексту и замене слов**, используя стандартную функцию Python `replace()`.  
Программа интерактивная — пользователь сам вводит текст и слова для замены, и сразу видит результат в терминале.  

Это отличный способ понять, как работают строки и словари в Python, а также потренироваться с основами программирования.
